# Проект HTML-сторінок

Цей репозиторій містить набір простих HTML-сторінок та прикладів для навчальних занять.

## Структура репозиторію

- `index.html` — головна сторінка.
- `articles.html`, `catalog.html`, `contacts.html`, `gallery.html`, `page-nav.html`, `registration.html` — окремі приклади сторінок.
- `lesson_3/` — папка з додатковими сторінками та стилями для уроку 3.
- `lesson-4/` — папка з прикладом для уроку 4.

## Як переглянути проект

1. Відкрийте файл `index.html` у браузері.
2. Або запустіть локальний HTTP-сервер у корені проекту та відкрийте адресу `http://localhost:5500` (або іншу, залежно від сервера).

## Примітки

- Файли CSS знаходяться в `lesson_3/css/style.css` та `lesson-4/style-css/style.css`.
- Всі сторінки повністю працюють як статичний HTML без серверної частини.
